# README

## 概要
`StorageMonitor_Portable.exe` は、USB メモリや任意のフォルダに入れてそのまま実行できる **ポータブル版** のストレージ使用率モニターです。
- 設定ファイルは実行ファイルと同じフォルダに保存されるため、フォルダごとコピーすれば別の PC でも同じ設定で動作します。
- インストーラや管理者権限は不要です。

## 必要環境
- Windows 11（64bit）

## 実行手順
1. `StorageMonitor_Portable.exe` を任意の場所にコピーします。 `（例:%USERPROFILE%\Desktop\StorageMonitor\`）
2. エクスプローラでファイルを **ダブルクリック** します。
3. 起動するとタスクトレイに常駐し、ドライブの使用率が表示されます。

## 設定
- 設定は `settings.json` に保存され、実行ファイルと同じフォルダに作成されます。
- 設定項目（自動起動、更新間隔、テーマ）はアプリの設定画面から変更できます。
- 設定をデフォルトに戻したい場合は、`settings.json` を削除するか、設定画面の **「設定をリセット」** ボタンを使用してください。

## 終了・バックグラウンド実行
- タスクトレイアイコンを **右クリック → 終了** でアプリを完全に終了します。
- ウィンドウの **× ボタン** を押すとカスタムダイアログが表示され、
  - 「バックグラウンドで実行」：ウィンドウを閉じてタスクトレイに残す
  - 「アプリを完全に終了」：即座に終了
  - 「キャンセル」：何もしない

## トラブルシューティング
- 起動しない場合は、**同じフォルダ内の `settings.json` を削除** して再起動してください。
- タスクトレイにアイコンが表示されない場合は、**ウイルス対策ソフトがブロックしていないか** をチェックしてください。
